Projects

  • Solar Controller Dashboard: Built a real-time monitoring dashboard for the ProStar PS-15M solar controller using Modbus RTU communication. Used Python and Flask to collect, decode, and visualize data from battery, load, and charge controllers. Designed the system for Raspberry Pi, with features for historical data logging, voltage and current tracking, and battery health analysis. Focused on accuracy, usability, and system stability.

  • Border Wait Time System: Developed a system to estimate border crossing wait times using Raspberry Pi and live camera feeds. Applied image processing techniques to detect and count vehicles in real time. Designed the hardware setup, implemented Python-based object detection, and tested accuracy under varied lighting and traffic conditions. Focused on improving reliability and responsiveness in data collection for transportation monitoring.

  • MXET Letter Display: Designed a system that displays different letters and lights up LEDs based on input audio frequency and amplitude. Used signal processing techniques to analyze real-time audio input and map specific frequency ranges and volume levels to corresponding visual outputs. Programmed microcontroller logic to control LED behavior and character display, with an emphasis on timing accuracy and response sensitivity.

  • Light-Reactive Crawling Microbug: Assembled and soldered a light-following robot using discrete components and analog signal control. Designed to move toward a light source using two LDRs, a transistor-based control system, and pulsed motor output. Built the full circuit on a perfboard, adjusted component values for sensor accuracy, and tested transistor behavior using oscilloscope and multimeter. Verified differential motor control based on light intensity for directional motion.
